九宫格计算思路

  • 利用控件的索引index计算出控件所在的行号和列号
  • 利用列号计算控件的x值
  • 利用行号计算控件的y值

定时任务

  • 方法1：performSelector
    // 1.5后自动调用self的hide方法
[self performSelector:@selector(hide) withObject:nil afterDelay:1.5];
  • 方法2：GCD
    //1.5秒后自动执行block里面的代码
 dispatch_after(dispatch_time(DISPATCH_TIME_NOW, (int64_t)(1.5 * NSEC_PER_SEC)), dispatch_get_main_queue(), ^{
     self.hudLbl.alpha = 0.0;
 });
  • 方法3：NSTimer
    //1.5s后自动调用self的hide方法
[NSTimer timerWithTimeInterval:1.5 target:self selector:@selector(hide) userInfo:nil repeats:NO];
//repeats如果为YES，意味着每隔1.5s就会调用一次self的hide方法

    常见问题

    • 项目里面的某个.m文件无法使用
      • 检查: Build Phases-> Compile Sources
    • 项目里面的某个资源文件(比如plist 音频等)无法使用
      • 检查: Build Phases -> Copy Bundle Resources

        模型

  • 什么是模型?
    • 专门用来存放数据的对象
    • 一般都是一些直接继承NSObject的纯对象
    • 内部会提供一些属性来存放数据

一个控件看不见有哪些可能？

  • 宽度或者高度其实为0
  • 位置不对（比如是个负数或者超大的数，已经超出屏幕）
  • hidden == YES
  • alpha <= 0.01
  • 没有设置背景色、没有设置内容
  • 可能是文字颜色和背景色一样

封装控件的基本步骤

  • 在initWithFrame:方法中添加子控件，提供便利构造方法
  • 在layoutSubviews方法中设置子控件的frame(一定要调用super的layoutSubviews)
  • 增加模型属性，在模型属性set方法中设置数据到子控件上
